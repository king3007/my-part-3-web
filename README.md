Project Overview
This repository contains the full website project for *Patrao’s Hair Salon*, developed for the WEDE5020 POE (Part 1–3).  
The website includes multiple pages, CSS styling, JavaScript functionality, responsive design, SEO optimisation, forms, and documentation.

The purpose of this project is to create a professional, functional, and visually appealing website for a small business.



 Website Goals & Objectives
- Provide information about the salon and its services.
- Create a clean, easy-to-navigate website structure.
- Display a gallery of hairstyles and salon photos.
- Allow customers to send enquiries and contact messages.
- Include maps showing salon locations.
- Implement JavaScript interaction and validation.
- Improve SEO performance using best practices.

---

 Key Features & Functionality

 HTML & CSS
- 5 main pages: Home, About, Gallery, Enquiry, Contact.
- External style.css linked to all pages.
- Consistent colour theme and typography.
- Fully responsive design using media queries.
- Clean structure using semantic HTML.

 JavaScript Enhancements
- Form validation for Enquiry and Contact pages.
- Interactive Leaflet map with location marker.
- Dynamic error messages.
- Improved user experience through interactivity.

 SEO Improvements
- Title tags & meta descriptions added.
- Alt text included for all images.
- Sitemap (sitemap.xml) created.
- robots.txt included for crawler instructions.
- Clean and consistent URL structure.
- Mobile-friendly layout for better ranking.
